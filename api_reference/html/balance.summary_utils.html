balance.summary_utils¶

Shared summary and diagnostics utilities for the Balance library.

Provides _build_summary() and _build_diagnostics(), which are shared by both the legacy Sample class and the new BalanceFrame workflow.

These functions accept plain DataFrames / Series instead of Sample objects, so callers can produce the same output without tight coupling to the Sample class hierarchy.
