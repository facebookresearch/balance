This page embeds the HTML rendered from the tutorial notebook
(`tutorials/balance_diff_diff_brfss.ipynb`). The official
import-balance.org build always has the optional `[did]` extra
installed, so the embedded notebook below renders normally there.
**For local docs builds**, `scripts/make_docs.sh` skips the brfss
tutorial nbconvert step with a notice when `diff_diff` is not
installed; if the iframe below shows a missing file / 404, install
the optional dependency and rerun the docs build:

```bash
pip install "balance[did]"
bash scripts/make_docs.sh -n
```

:::
